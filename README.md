# Typo - Modern Focus-Oriented Typing Test

Typo is a clean, modern, and minimalist typing test application designed for focus and performance tracking.

## Features

- **Focus Mode**: UI elements fade away as you start typing.
- **Modern UI**: Borderless design with a choice of Light and Dark themes.
- **Live Tracking**: Real-time WPM, Accuracy, and Time tracking.
- **Performance Chart**: Visual history of your WPM during each session.
- **Customizable Modes**: Choose between word-count and time-based tests.
- **Keyboard First**: Support for shortcuts like `Shift + Enter` to restart.

## Tech Stack

- **React** (v19)
- **TypeScript**
- **Vite**
- **Framer Motion** (for smooth animations)
- **Lucide React** (for icons)
- **Vanilla CSS** (for styling)

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the development server:
```bash
npm run dev
```

### Deployment (GitHub Pages)

This project is configured for easy deployment to GitHub Pages.

1. Update the `homepage` field in `package.json` to your GitHub Pages URL:
   ```json
   "homepage": "https://<your-username>.github.io/<your-repo-name>",
   ```
2. Deploy the application:
   ```bash
   npm run deploy
   ```
